# 論文紹介"Opponent and bidirectional control of movement velocity in the basal ganglia"(基底核における運動速度の拮抗的および双方向的制御)

## 文献情報

### 著者: Eric A. Yttri1 and Joshua T. Dudman

### 雑誌名: Nature

### 巻・号: 533

### ページ: 402–406

### 出版年: 2016

## Keyword

- 大脳基底核(BG)
  - 線条体（STR）:大脳基底核の入力部の１つ、大脳皮質から入力を受け、黒質網様部、淡蒼球内節、淡蒼球外節に投射
  - 黒質緻密部(SNc):主に大脳基底核の線条体にドーパミンを投射
  - 黒質網様部(SNr):大脳基底核の出力部の１つ、主に視床に出力
  - 淡蒼球内節(GPi):大脳基底核の出力部の１つ、主に視床に出力
  - 淡蒼球外節(GPe):間接経路の中継部、淡蒼球内節,視床下核に投射
  - 視床(The thalamus)：大脳基底核の出力先
  - 視床下核(subthalamic nucleus(STN)):大脳基底核の入力部の１つ、大脳皮質、淡蒼球外節から入力を受け、淡蒼球内節、淡蒼球外節、黒質網様部に対して、投射
- ドーパミン(DA)
- 大脳皮質(Cortex):大脳基底核の入力元
- 直接経路 STR⇒GPi/SNr
- 間接経路 STR⇒GPe⇒STN⇒GPi
- ChR2(チャネルロドプシン-2):緑藻由来の光感受性タンパク質。青色光（約470 nm）を受けると開き、Na+、K+、Ca2+、H+といった陽イオンが細胞膜を通過します。これにより、細胞膜の脱分極が引き起こされ、神経細胞などの興奮性細胞で活動電位が発生する

### イントロダクション
- 目的のある行動には、目標の選択（左または右など）と実行パラメータ（速度など）が必要 
- 例えば鳥のさえずりでは、離散的かつ順序的な行動（音節の並び）と音の高さ（ピッチ）の両方が、大脳皮質－基底核経路における強化学習によって制御される。
- 脊椎動物において、線条体は基底核の主要な入力核であり、直接路と間接路は分子レベルで異なる2種類の投射ニューロン（MSN）から主に構成される。
  - 直接路ニューロン（direct striatonigral(dMSN)）= 直接路
  - 間接路ニューロン（ indirect striatopallidal (iMSN)）= 間接路
- dMSNの持続的活性化は運動を促進させ、iMSNの持続的活性化は運動を抑制する。
- 皮質からdMSN・iMSNへのシナプスにおいて、それらのシナプスが強化されたり、弱化されたりすることで目標を達成するような行動の選択を支えていると考えられる
- 直接路ニューロンを刺激すると自己刺激（実験的に、マウスに「レバーを押すと直接路ニューロンが刺激される」という仕組みを与えると、マウスは自分から繰り返しレバーを押し続ける）や選択行動の偏り（2つのレバーがあるとき、一方に「直接路刺激」が結びついていれば、そのレバーを選びやすくなる。）が起こる
- MSNの活動しただけで、その後の行動に持続的で特定の変化を起こすことを直接示した証拠はまだ少ない
- 目的：大脳基底核の腕の活動などの身体の活動に依存する可逆性が速度などの運動のkinematicsに学習的変化を生み出すのに十分かどうかを確かめる

### 手法
- 実験内容
  - 8匹のマウスで実験
  - 図１
    - Channelrhodopsin-2 (ChR2)：光（青色光）で神経を活動させることができるタンパク質をdMSNまたはiMSNを遺伝子に組み込む→光を当てると、埋め込まれたニューロンが発火する
    - マウスの頭を側面に取り付けられたジョイスティックと給水口の前に固定
    - 光ファイバーを線条体に埋め込む
    - マウスは液体報酬を受け取るためにジョイスティックで前肢(pull or push)を動かし基準距離を超えると１秒後に報酬
    - 試行感覚は3秒（手掛かりなし＝音が鳴ったりしない）
    - タスクを達成するモチベーションを保つために、基準距離を越えた全ての行動に遅延報酬を与えた
    - 自発的な行動の際にMSNの活動が運動パラメータにどのような影響を与えるか調べるために、マウスの動作の中でも特に速い動き（速度の上位３分の１）のときに光刺激を入れた
    - 光刺激の強度は運動への直接的な影響を起こさないくらい
    - 動きが始まってから 15ミリ秒以内に光を当てる
    - 450ミリ秒間（＝だいたい1回の動作の長さに匹敵する時間）刺激を続けた
    - 緑の三角は試行の開始、黄色の四角は報酬
    - dは棒の最大振幅、最大速度と線条体の発火の継続時間（青色の線が光を当てていた時間）のヒストグラム
    - eは整列した線条体の平均反応。緑の線が試行の開始
    - fは１試行中の光刺激中の線条体の個体群のラスタープロット。オレンジの線は報酬を与えたのかor刺激を与えたタイミングなのか？
- 図２
  - 光刺激を加えたときの速度などのパラメータの変化
  - aは50回の施行のうち、dMSNまたはiMSNが刺激が受けた場合と受けなかった場合（偽セッション）のピーク速度の差(Δ速度)。その後の 回復期（光なし） のデータも併せて載せている。
  - 累積的に増加し、ベースラインに戻る
  - bは偽と刺激の場合の試行間隔と舐める速度のヒストグラム（黒が偽、青がdMSN、赤がiMSNに対応している）
    - 縦軸は正規化してある（セッション数の違いの関係）
    - 最も速い動きの際の短時間のdMSN刺激もiMSN刺激も、報酬の期待と消費の間に試行開始率や舐め速度に変化をもたらさない
    - 言えること:MSNの活性化が、運動や動機付けの一般的な変化なしに、運動パラメータの持続的な変化を生み出すのに十分である
  - cは刺激されたときの移動パラメータの変化の母集団平均
    - dMSN刺激はピーク速度の有意な増加
    - iMSN刺激はピーク速度の有意な減少
    - 振幅、持続時間、曲がりは変化しない
    - 訓練された動物は、変化するタスク要件に合わせて振幅を調整できる（extended data 3）
  - dはaの母集団平均バージョン。影は標準偏差
    - 刺激が繰り返されることで、速度が緩やかで累積的な変化を起こす
    - 回復期でも変化→刺激による速度分布の変化は、回復ブロック中に徐々に刺激前のベースラインに戻るまで、数十回の試行で持続する（どちらの場合も同じ時間かかる）
    - セッション全体を通して、速度分布の分散に変化なし
  - ここまでの結果だと「dMSN刺激 → 速くなる」「iMSN刺激 → 遅くなる」という単純な対抗関係（opponency）が見えていたが、学習の観点からは不十分
  - 強化学習では「報酬を増やすために行動を変える」ことが重要→行動が「速くなる」でも「遅くなる」でも、報酬につながるならどちらでも学習できるはず。
  - efはcdの最遅バージョン
    - dMSN刺激とiMSN刺激の両方とも速度に対して、逆の効果をもたらす
    - dMSNは累積的な低下を引き起こす
    - iMSN刺激は速度の累積的な増加を引き起こす
    - 目的に対して、速度に対する役割が反対になる
    - 直接路と間接路は「片方が加速／もう片方が減速する」だけの対抗関係ではなく、直接路と間接路は 双方向に運動速度を調整する（bidirectional control） 働きを持つ
- 図３について
  - 結果を踏まえたモデル作成：The MeSH learning rule
    - $M$:ある運動パラメータ$m$に対応する正規分布。各試行$i$において、その分布からランダムに $m_i$が選ばれる
    - $\bar{M}$:分布の平均
    - $r$:報酬。この報酬によって、分布$M$の平均は報酬に応じて変化する
    - $S$: dMSN の場合は +1、iMSN の場合は −1、刺激なしでは 0
  - aはThe MeSH learning ruleを使ったシミュレーションにおけるdMSNとiMSN刺激における平均のピーク速度
    - 試行中のランダムな刺激は検出可能な移動速度の変化を生じない
    - MeSHは「報酬信号（ドーパミン）」と「線条体ニューロン（MSN）の活動」が一緒に起こるとシナプス可塑性が起きる＝ドーパミン入力（報酬予測誤差など）が「これが価値のある動きだよ」と伝え、MSNの活動が「今この動きをしているよ」と伝えることで、この2つの一致がその動きのシナプス結合が強化する
    - 先行研究:線条体（striatum）を直接刺激すると、動物はドーパミン受容体をブロックしても 自分からその刺激を求める行動を繰り返す
    - 先行研究では対照的な結果が出ていたが、そちらは慢性的に刺激を与えていた
    - 運動速度の変化を引き出すための刺激にドーパミンが必要かどうかが疑問 
  - bはドーパミン受容体拮抗薬（ドーパミンが受容体に結合するのを阻害 する薬）下で最速３分の１で刺激を与えた時のΔ速度
    - 注射後、dMSNまたはiMSNのいずれかの刺激後の速度の持続的な変化が排除されることがわかった
  - cはドーパミン拮抗薬投与後の運動パラメータ分布
    - 全てのパラメータが刺激の有無で変わらない
    - 言いたいこと:ドーパミン拮抗薬は、dMSN刺激およびiMSN刺激の両方について、刺激効果の大きさを有意に低下させた
  - dはさまざまな条件下における刺激に対する速度の変化の結果
    - openloop:動物の行動や神経活動とは無関係に、外部から一方的に入力
    - 上２つが有意差あり
- 先行研究と今回の結果の比較:運動の速度は皮質神経節と大脳基底核の両方の出力によって決定されるという仮定と一致
- 主張:光刺激が促進と抑制の両方を増強すると仮定し、選択的刺激が偏った変化をもたらすのに対し、非選択的刺激は正味の変化を生じさせない
- 図4について
  - 皮質ー大脳基底核モデル（extended 6）
  - 皮質と大脳基底核の出力で速度が決まる
  - Corticostriatal circuit model（PDF参照）
  - aはこのモデルを用いた際のシミュレーション結果
  - bはMSNsの活動と運動速度の関連性（傾き）が、シミュレーションを通じてどれだけ強くなったかを示す。三角形は傾きの平均値を示す。円は皮質線条体シナプスの重みの平均変化量。→　平均運動速度は皮質-線条体シナプスの平均的な重みに比例する
  - cは横軸がピーク運動速度で縦軸が運動中の平均発火率
    - 傾きが大きければ、速く動くほど強く発火する。
    - シミュレーションで35個のユニットのデータを記録し、その中でも代表的なものを３つ図示している
  - dは横軸は非刺激試行でのdMSNの活動における運動速度の変化で、縦軸が光刺激を当てたときのベースライン発火率の変化
    - 光刺激時の発火率が、ベースライン（非刺激時）と比べて何％変化したかを図示
    - 光刺激は単発反応ではなく、可塑性を通してニューロンにおける速度のチューニングを行う
